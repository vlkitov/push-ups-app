# Push-ups App - Настройка проекта

## 📋 Необходимые настройки в Info.plist

Добавьте следующие ключи в `Info.plist`:

```xml
<key>NSCameraUsageDescription</key>
<string>Камера используется для автоматического подсчета отжиманий через отслеживание расстояния до вашего лица</string>

<key>NSFaceIDUsageDescription</key>
<string>TrueDepth камера используется для точного определения расстояния во время отжиманий</string>
```

## 🎯 Необходимые Capabilities

В настройках проекта (Target → Signing & Capabilities) убедитесь что включено:
- **Camera** (автоматически при использовании ARKit)

## 🚀 Принцип работы

1. **Калибровка**: При запуске тренировки пользователь занимает верхнюю позицию (руки выпрямлены). Приложение собирает 30 кадров (~1 секунда) и запоминает начальное расстояние от камеры до лица.

2. **Отслеживание**: ARKit отслеживает 3D позицию лица в реальном времени. Используется евклидово расстояние (√(x²+y²+z²)), которое работает независимо от ориентации устройства.

3. **Определение фазы**:
   - **Вниз**: Когда лицо приблизилось к камере на 20см от базовой линии (расстояние уменьшилось)
   - **Вверх**: Когда лицо вернулось к исходному расстоянию (±8см от базовой линии)

4. **Подсчет**: Полный цикл вниз→вверх засчитывается как одно отжимание

### 📐 Почему евклидово расстояние?

При использовании только Z-координаты из матрицы трансформации, детектор работает только когда телефон вертикально. Когда телефон лежит на полу горизонтально, оси координат поворачиваются, и Z больше не показывает расстояние до лица.

**Решение**: Вычисляем полное 3D расстояние от камеры (точка 0,0,0) до позиции лица:
```swift
let distance = simd_length(simd_float3(x, y, z))
```

Это работает в любой ориентации устройства!

## 🎨 Особенности реализации

- **Сглаживание данных**: Используется скользящее среднее для уменьшения шума
- **Защита от дребезга**: Минимальный интервал между повторениями 0.5 сек
- **Машина состояний**: Точное определение фаз отжимания
- **Тактильная обратная связь**: Haptic Feedback при каждом повторении
- **Визуальный прогресс**: Индикатор прогресса текущего отжимания

## 🔧 Настройка параметров

В файле `PushUpDetector.swift` можно настроить:
- `downThreshold: Float = 0.20` - порог для нижней точки (20см от базовой линии)
- `upThreshold: Float = 0.08` - порог для верхней точки (8см от базовой линии)
- `minimumRepInterval: TimeInterval = 0.5` - минимальное время между повторениями
- `smoothingWindowSize = 5` - размер окна сглаживания

**Рекомендации по настройке:**
- Увеличьте `downThreshold` для более глубоких отжиманий (строже)
- Уменьшите для начинающих или разминки
- `upThreshold` влияет на то, насколько нужно выпрямить руки
